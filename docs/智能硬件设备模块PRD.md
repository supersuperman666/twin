# 智能硬件设备模块 PRD

## 1. 模块定位

智能硬件模块作为慢病管理小程序的设备接入入口。本期重点完成设备关联入口、我的设备列表、华为账号授权静态页、重点睡眠/血氧设备蓝牙绑定静态流程，以及已绑定设备解绑能力。

本期不做真实蓝牙连接、真实华为账号授权、真实设备数据同步、复杂设备详情和采集数据记录。

## 2. 页面范围

| 页面 | 路径 | 说明 |
| --- | --- | --- |
| 设备关联页 | `/pages/device/index/index` | 按分类展示可添加设备，支持绑定/同步入口 |
| 我的设备页 | `/pages/device/my/index` | 展示已绑定设备，支持查看同步状态和解除绑定 |
| 华为账号授权静态页 | `/pages/device/huawei-auth/index` | 参考华为账号登录页，仅展示，不真实登录 |
| 蓝牙连接页 | `/pages/device/bluetooth/index` | 重点设备进入，展示三步绑定说明和立即绑定按钮 |
| 连接中静态页 | `/pages/device/connecting/index` | 点击立即绑定后进入，展示静态连接中状态 |

## 4. 核心交互

### 4.1 设备关联页

- 默认选中睡眠分类。
- 点击左侧分类，右侧设备列表切换。
- 设备卡片展示设备图标、设备名称、说明、接入来源和操作按钮。
- 未绑定设备按钮展示为“去绑定”或“去同步”。
- 已绑定设备在设备关联页展示“已绑定”，不在此页直接解绑。
- 用户需要管理或解绑设备时，进入“我的设备页”。

### 4.2 我的设备页

入口：

| 来源 | 展示状态 | 入口文案 | 去向 |
| --- | --- | --- | --- |
| 首页-我的设备模块 | 已绑定 X 台 | 查看设备 | 我的设备页 |
| 我的页/健康档案 | 我的设备 | 查看设备 | 我的设备页 |
| 记录页设备入口 | 已有绑定设备时 | 查看设备 | 我的设备页 |

页面定位：

- 展示用户已绑定设备。
- 让用户确认设备型号和最近同步状态。
- 支持解除绑定。
- 若用户没有任何绑定设备，展示空状态并引导去设备关联页。

设备卡片字段：

| 字段 | 示例 | 说明 |
| --- | --- | --- |
| 设备名称 | 血氧仪 | 面向患者的通用设备名称 |
| 设备型号 | ZG-P11H | 设备型号或来源型号 |
| 最近同步 | 今天 07:45 | 最近一次同步时间；无同步数据时展示“暂无同步记录” |
| 同步状态 | 正常 / 同步中 / 同步异常 | 用于提示是否需要处理 |
| 操作 | 解除绑定 | 解绑入口 |

通用设备示例：

| 设备名称 | 设备型号 | 最近同步 | 同步状态 |
| --- | --- | --- | --- |
| 血糖仪 | SN-GM01 | 今天 08:12 | 正常 |
| 血压计 | OMRON-BP01 | 昨天 21:30 | 正常 |
| 血氧仪 | ZG-P11H | 今天 07:45 | 正常 |
| 睡眠/呼吸设备 | ZG-M11A | 今日报告生成中 | 同步中 |

空状态：

```text
标题：还没有绑定设备
说明：绑定设备后，可自动同步部分健康数据，减少手动记录。
按钮：去绑定设备
```

点击“去绑定设备”进入设备关联页。

解除绑定流程：

1. 用户点击设备卡片中的“解除绑定”。
2. 弹出挽留确认弹窗。
3. 用户点击“继续保留”则关闭弹窗，不执行解绑。
4. 用户点击“确认解绑”后，执行设备解绑。
5. 解绑成功后刷新我的设备列表，并同步更新首页设备状态。

挽留弹窗建议文案：

```text
标题：确认解除绑定？
说明：解绑后，该设备将不再自动同步健康数据，可能影响健康风险提示和医生查看最近记录。
主按钮：继续保留
次按钮：确认解绑
```

解绑规则：

- 解绑只解除后续数据同步和设备关系。
- 已同步的历史健康数据和医生处理记录不删除。
- 解绑后需记录操作人、操作时间、设备 ID、设备型号和解绑来源。
- 若解绑后没有任何设备，首页“我的设备”模块回到未绑定状态。

### 4.3 华为授权流程

- 非重点蓝牙设备点击“去同步”后统一进入华为账号授权静态页。
- 页面仅用于演示华为账号登录视觉，不提交账号密码。
- 点击登录提示“当前为授权流程演示页”。

### 4.4 蓝牙绑定流程

- 血氧指环 ZG-P11H、血氧指环 ZG-P11/G、雷达睡眠监测仪 ZG-M11A、雷达睡眠监测仪 ZG-M11B 点击“去绑定”进入蓝牙连接页。
- 蓝牙连接页展示三步：
  1. 确认硬件处于工作状态。
  2. 打开手机蓝牙。
  3. 在本小程序的绑定部分点击绑定，手机自动查找设备。
- 点击“立即绑定”进入连接中静态页。
- 本期连接中页面仅展示静态状态，不做真实蓝牙扫描；为了演示解绑闭环，进入连接中页后可在本地 mock 标记为已绑定。

## 5. 验收标准

- 智能硬件页左侧展示 6 个分类：睡眠、血糖、血压、血氧、体重、运动。
- 血氧指环 ZG-P11H、血氧指环 ZG-P11/G 展示在血氧分类。
- 雷达睡眠监测仪 ZG-M11A、雷达睡眠监测仪 ZG-M11B 展示在睡眠分类。
- 上述四个重点设备点击后进入蓝牙连接页。
- 蓝牙连接页点击“立即绑定”进入连接中静态页。
- 其他设备点击后进入华为账号授权静态页。
- 已绑定设备在设备关联页展示“已绑定”，不直接解绑。
- 我的设备页展示已绑定设备列表，字段包含设备名称、设备型号、最近同步、同步状态和解除绑定。
- 点击“解除绑定”弹出挽留确认弹窗，点击“继续保留”不解绑，点击“确认解绑”后执行解绑。
- 解绑成功后刷新我的设备页；若没有任何绑定设备，展示空状态并引导去设备关联页。
