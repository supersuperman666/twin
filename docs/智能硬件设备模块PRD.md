# 智能硬件设备模块 PRD

## 1. 模块定位

智能硬件模块作为慢病管理小程序的设备接入入口。本期重点完成设备分类展示、设备绑定入口、华为账号授权静态页、重点睡眠/血氧设备蓝牙绑定静态流程，以及已绑定设备解绑能力。

本期不做真实蓝牙连接、真实华为账号授权、真实设备数据同步、设备详情和采集数据记录。

## 2. 页面范围

| 页面 | 路径 | 说明 |
| --- | --- | --- |
| 智能硬件 | `/pages/device/index/index` | 按分类展示可添加设备，支持绑定/同步/解绑入口 |
| 华为账号授权静态页 | `/pages/device/huawei-auth/index` | 参考华为账号登录页，仅展示，不真实登录 |
| 蓝牙连接页 | `/pages/device/bluetooth/index` | 重点设备进入，展示三步绑定说明和立即绑定按钮 |
| 连接中静态页 | `/pages/device/connecting/index` | 点击立即绑定后进入，展示静态连接中状态 |

## 4. 核心交互

### 4.1 智能硬件页

- 默认选中睡眠分类。
- 点击左侧分类，右侧设备列表切换。
- 设备卡片展示设备图标、设备名称、说明、接入来源和操作按钮。
- 已绑定设备按钮展示为“解绑”。
- 点击“解绑”弹出确认框，确认后移除本地绑定状态。

### 4.2 华为授权流程

- 非重点蓝牙设备点击“去同步”后统一进入华为账号授权静态页。
- 页面仅用于演示华为账号登录视觉，不提交账号密码。
- 点击登录提示“当前为授权流程演示页”。

### 4.3 蓝牙绑定流程

- 血氧指环 ZG-P11H、血氧指环 ZG-P11/G、雷达睡眠监测仪 ZG-M11A、雷达睡眠监测仪 ZG-M11B 点击“去绑定”进入蓝牙连接页。
- 蓝牙连接页展示三步：
  1. 确认硬件处于工作状态。
  2. 打开手机蓝牙。
  3. 在本小程序的绑定部分点击绑定，手机自动查找设备。
- 点击“立即绑定”进入连接中静态页。
- 本期连接中页面仅展示静态状态，不做真实蓝牙扫描；为了演示解绑闭环，进入连接中页后可在本地 mock 标记为已绑定。

## 5. 验收标准

- 智能硬件页左侧展示 6 个分类：睡眠、血糖、血压、血氧、体重、运动。
- 血氧指环 ZG-P11H、血氧指环 ZG-P11/G 展示在血氧分类。
- 雷达睡眠监测仪 ZG-M11A、雷达睡眠监测仪 ZG-M11B 展示在睡眠分类。
- 上述四个重点设备点击后进入蓝牙连接页。
- 蓝牙连接页点击“立即绑定”进入连接中静态页。
- 其他设备点击后进入华为账号授权静态页。
- 已绑定设备在智能硬件页展示“解绑”，点击后可确认解绑。
