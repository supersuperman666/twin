# 患者端-记录页SPEC

> 来源：主 PRD《执行任务闭环PRD》、患者端总览 SPEC、旧版小程序 PRD 记录/快速记录章节与记录指标模块 PRD。本文档聚焦记录页结构、锚点分组和记录入口。

## 1. 本期目标

P0 必须跑通：

```text
记录页
  -> 主动补录
  -> 查看历史
  -> 查看各类记录分组
  -> 进入具体记录表单
```

## 2. 页面定位

记录页是通用记录工具页，不替代方案页。

- 方案页偏“按任务执行”
- 记录页偏“主动补录和查看历史”

## 3. 页面结构

采用：

- 日期选择
- 吸顶锚点导航
- 分区内容结构

锚点建议：

- 全部
- 基础指标
- 血氧呼吸
- 睡眠
- 症状
- 用药

## 4. 分区说明

### 4.1 基础指标

- 血糖
- 血压
- 体重

### 4.2 血氧呼吸

- SpO2
- 呼吸频率
- 活动后/静息后等记录

### 4.3 睡眠

- 睡眠报告卡片
- 睡眠时长
- AHI / ODI / 最低血氧摘要

复杂事件指标进入睡眠分析二级页。

### 4.4 症状

- 症状记录卡片
- 症状严重程度
- 发生时间和备注

### 4.5 用药

- 用药执行记录
- 是否漏服
- 漏服原因

## 5. 记录类型与字段概览

| 类型 | 关键字段 |
| --- | --- |
| 血糖 | 血糖值、测量时点、时间、目标范围、备注 |
| 血氧 | SpO2、脉率、时间、场景、备注 |
| 血压 | 收缩压、舒张压、时间、备注 |
| 睡眠 | 睡眠报告卡片、睡眠时长、AHI、ODI、最低血氧 |
| 用药 | 药品、剂量、服用时间、是否漏服、漏服原因 |
| 症状 | 症状项、严重程度、时间、备注 |

## 6. 交互要求

- 支持手动录入、编辑、删除
- 记录后刷新首页待办和预警
- 数值异常时即时提示建议复测/联系医生
- 保留异常值，但需二次确认

## 7. 跳转关系

| 入口 | 去向 |
| --- | --- |
| 快速记录卡 | 记录表单页 |
| 睡眠卡片 | 睡眠报告页 |
| 历史记录 | 记录详情页 |
| 扣分项/预警联动 | 对应详情页 |

## 8. 一句话总结

**记录页是患者端的通用工具区，用来补录、回看和细看，不承担今日执行主入口。**
