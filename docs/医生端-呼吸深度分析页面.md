## 1 文档信息

### 1.1 文档用途
明确医生PC端慢阻肺深度分析页面功能定义、数据逻辑、页面布局、交互规则、展示样式，统一设计、开发、测试标准。

### 1.2 数据来源与能力边界

| 设备 | 型号 | 硬件直接采集 | 软件衍生指标 | 本页定位 |
|------|------|-------------|-------------|---------|
| 医用脉搏血氧仪 | ZG-P11H / ZG-P11G | SpO₂连续采样、脉率连续采样 | 平均血氧、最低血氧、ODI、氧减事件次数与分布、Ts90%、血氧趋势图、脉率趋势图 | 全天血氧监测与低氧风险评估（慢阻肺核心关切） |
| 睡眠呼吸监测系统 | ZG-M11A / ZG-M11B | 呼吸运动波形（UWB雷达）、体动、心率、SpO₂（内置P11指环）、体位（仅M11A） | AHI、呼吸暂停/低通气分型计数、睡眠分期（AI推算）、睡眠效率、鼾声摘要 | **不在本页展示**，完整睡眠呼吸事件分析归属「睡眠深度分析」Tab |
| 患者端自评 | - | CAT慢阻肺评估测试（每日填报）、mMRC呼吸困难指数（定期评估） | - | 主观症状评估 |

**关键澄清**：
- P11指环是全天佩戴设备，可24小时连续采集SpO₂和脉率，不限于夜间使用
- 慢阻肺患者白天静息血氧、活动后血氧下降同样是核心关切，本页不限于夜间数据
- P11单独使用时输出 SpO₂ + 脉率 + 初筛系统衍生指标（ODI、氧减事件等），不输出AHI
- M11系统内置P11指环采集血氧，雷达采集呼吸运动，两者联合才能输出AHI和呼吸事件分型
- 本页**不展示AHI、呼吸暂停分型、低通气明细、睡眠分期、体动、鼾声**，这些指标仅在「睡眠深度分析」Tab展示
- 本页**不展示睡眠呼吸事件摘要和参考性睡眠分期**，这些属于睡眠专项指标，慢阻肺Tab不混杂睡眠诊断级数据

### 1.3 设计开发原则
- 临床数据合规：系统提供风险提示，所有诊断和处方决策需医生确认
- 设备能力诚实：衍生指标标注来源和精度等级，不伪造超出硬件能力的指标
- 慢阻肺临床聚焦：围绕血氧低氧和氧减负担展开，不混杂睡眠诊断级指标
- 页面无冗余：与睡眠深度分析Tab严格分界，避免数据重复展示
- 多状态场景全覆盖：数据不足、设备异常、轻中度异常均有明确交互规则

## 2 需求概述

本页面为患者详情页下专属慢阻肺深度分析模块，整合**连续血氧监测数据**与**患者主观自评问卷**，围绕低氧、氧减负担、症状波动开展综合研判。

核心业务价值：
- 评估慢阻肺患者血氧低氧风险与氧减负担（全天佩戴P11指环，不限于夜间）
- 预警急性加重风险（持续低氧是慢阻肺急性加重前兆）
- 辅助医生制定氧疗与用药干预方案
- 客观血氧数据与主观症状相互参照

**本页不覆盖**：睡眠呼吸事件诊断分析（AHI、呼吸暂停分型、睡眠分期、体动、鼾声等）归属「睡眠深度分析」Tab

## 4 与睡眠深度分析Tab的边界

### 4.1 分界原则

慢阻肺深度分析和睡眠深度分析是两个独立一级Tab，各自聚焦不同临床问题：

| 维度 | 慢阻肺深度分析 | 睡眠深度分析 |
|------|---------------|-------------|
| 临床关切 | "这个慢阻肺患者低氧和氧减负担如何，是否需氧疗干预？" | "这个睡眠呼吸障碍患者呼吸事件和夜间低氧是否需进一步处理？" |
| 核心指标 | 血氧均值、最低血氧、ODI、Ts90%、低氧时长、低氧事件次数 | AHI、呼吸暂停/低通气分型、睡眠分期、体动、鼾声 |
| 数据来源 | P11指环连续血氧监测 + 患者问卷 | M11系统完整睡眠呼吸监测 + P11血氧 |
| 趋势图表 | 血氧时序曲线、近7/14天低氧趋势 | 多晚AHI/ODI趋势、单夜多轨详情 |
| 目标人群 | 有慢阻肺标签的患者（不论是否有睡眠呼吸障碍） | 有睡眠呼吸障碍标签的患者 |

### 4.2 共病患者的衔接

当患者同时有「慢阻肺」和「睡眠呼吸障碍」标签时，两个Tab均展示，但数据不重复：

- 慢阻肺Tab：展示低氧摘要（均值、最低值、ODI、Ts90%、低氧时长）
- 睡眠Tab：展示完整呼吸事件分析（AHI、呼吸暂停分型、低通气、睡眠分期）
- 慢阻肺Tab底部提供「共病睡眠关联」卡片：摘要展示ODI+最低血氧 + 【进入睡眠深度分析】跳转入口
- 睡眠Tab底部提供「慢阻肺低氧联动」卡片：摘要展示低氧指标 + 【进入慢阻肺深度分析】跳转入口

**无共病标签的患者不显示关联入口卡片。**

### 6.4 慢阻肺深度AI智能分析

固定4个分析维度，基于当前监测数据与问卷数据**模板填充**生成文案。

#### 6.4.1 整体缺氧风险评估

模板格式：
> 患者连续血氧均值{avgSpo2}%，最低血氧{minSpo2}%，Ts90%={ts90}%，ODI={odi}次/小时，整体缺氧风险判定为{riskLevel}。

判定规则：
- avgSpo2≥95% 且 minSpo2≥90% 且 ts90<5% → "正常"
- avgSpo2<95% 或 minSpo2<90% 或 ts90≥5%-10% → "轻度"
- avgSpo2<93% 或 minSpo2<85% 或 ts90≥10% → "重度"

#### 6.4.2 氧减负担评估

模板格式：
> ODI={odi}次/小时，低氧事件{hypoxEventCount}次，累计低氧时长{hypoxDuration}分钟，氧减负担{burdenLevel}。

判定规则：
- ODI<5 且 hypoxDuration=0 → "轻微"
- ODI 5-10 或 hypoxDuration 10-30分钟 → "中度"
- ODI≥10 或 hypoxDuration≥30分钟 → "重度"

#### 6.4.3 急性加重风险预警

模板格式：
> 累计低氧时长{hypoxDuration}分钟，Ts90%={ts90}%，重度低氧频次偏高；近{recentDays}天CAT评分变化{catMcidChange}分，mMRC评级变化{mmrcChange}级，慢阻肺急性加重风险处于{riskLevel}水平。

判定规则（综合客观+主观双维度）：
- ts90<5% 且 hypoxDuration<10分钟 且 CAT/MRC无上升趋势 → "低"
- ts90 5%-10% 或 hypoxDuration 10-30分钟 或 CAT近期上升≥2分(MCID) → "中"
- ts90≥10% 或 minSpo2<85% 或 mMRC近期上升≥1级 → "高"

主观趋势判定规则：
- CAT近期变化：取最近2次CAT评分差值，≥2分↑标注"有临床意义上升趋势"，≤-2分↓标注"有改善趋势"
- mMRC近期变化：取最近2次mMRC评级差值，≥1级↑标注"症状加重"，≤-1级↓标注"症状改善"
- 主观趋势与客观低氧叠加时，风险等级取更严重者（保守原则）

#### 6.4.4 主观症状+综合干预建议

模板格式：
> CAT评分{catScore}分（{catImpactLevel}），mMRC评级{mmrcGrade}级（{mmrcDesc}）；{interventionAdvice}。

### 7.3 判定优先级规则

当同一指标的不同等级边界值重叠时：
- 取更严重的等级（保守原则，临床安全优先）
- 如：ODI=10 按橙色上限还是红色下限？→ 取红色（≥10即红色）

### 7.4 多指标综合评级

页面整体风险评级取各指标中最严重的等级：
- 所有指标均为绿色 → 正常
- 任一指标为橙色 → 轻度异常
- 任一指标为红色 → 重度异常

## 8 指标计算时间窗规范

### 8.1 统一计算周期
P11指环全天佩戴，血氧类指标按**监测时段**计算。每个监测时段为设备连续采集的一段完整数据（如夜间22:00-07:30、日间08:00-18:00），由设备自动识别并生成报告。

默认展示最近一个完整监测时段的数据。全天数据可分段查看，每个时段独立计算指标。

### 8.2 单项指标计算规则

| 指标 | 计算规则 |
|------|---------|
| 平均血氧 | 监测周期内全部有效采样点算术平均值 |
| 最低血氧 | 周期内SpO₂数值最小值 |
| 低氧事件 | SpO₂低于90%且持续时长≥10秒，判定单次事件 |
| 低氧总时长 | 全部低氧事件持续时间累加求和 |
| Ts90% | (SpO₂<90%的总时长) ÷ (有效监测总时长) × 100% |
| ODI | 总氧减事件数(SpO₂下降≥3%或≥4%) ÷ 有效监测小时数 |

### 8.3 特殊场景处理

| 场景 | 处理规则 |
|------|---------|
| 监测时长不足4小时 | 按实际有效时长计算，页面标注真实时长和"监测时长不足"提示 |
| 日间短时数据 | 按实际时段计算并标注时段类型（日间/夜间），参与风险评级 |
| 设备信号中断 | 缺失段不计入采样，页面标注中断时段和时长 |

## 11 交互与空状态统一规则

| 场景 | 交互规则 |
|------|---------|
| Tab切换 | 局部刷新页面内容，不整页重载 |
| 空白数据 | 全页面统一文案「暂无数据」 |
| 风险配色 | 固定不变，所有模块统一识别标准 |
| AI分析文案 | 实时调取最新监测+问卷数据模板填充生成 |
| 问卷板块 | 常驻页面，不因任何条件隐藏 |
| 设备不支持模块 | 直接隐藏，不展示空态占位 |
| 设备支持但暂无数据 | 展示空态文案 |
| 参考指标 | 灰色边框+标注"参考"，不使用风险色条 |
| 趋势图缺失时段 | 显示断点和缺失提示，不用0值补齐 |
| 监测时长不足 | 标注真实时长+"监测时长不足"提示条 |
| 设备信号中断 | 标注中断时段和时长 |
| 多晚趋势不足 | 提示"有效报告不足，暂无法形成趋势" |

### 页面级空状态

|------|------|
