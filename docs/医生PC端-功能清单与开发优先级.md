# 医生 PC 端 — 功能清单与开发优先级

> 版本：V1.0 | 更新：2026-05-26 | 来源：代码实现梳理（apps/doctor-pc/src/main.js）

---

## 一、核心业务流程

```
医生登录
   │
   ▼
工作台（待办 / 高风险患者 / 统计概览）
   │
   ▼
患者管理列表（搜索 / 筛选 / 风险排序 / 标签）
   │ 点击患者
   ▼
患者详情 360（头卡 + 11 个子 Tab）
   │
   ├─→ 数字孪生融合分析（综合健康分 / 共病链路 / 管理建议）
   ├─→ 血糖 / 睡眠 / 慢阻肺 深度分析
   ├─→ 健康档案 / 健康筛查
   ├─→ 预警记录 → 预警中心处置 → 安排复测 / 随访 / 调方案
   ├─→ 管理方案 → 草稿 → 下发 → 执行 → 调整
   └─→ 随访计划 → 创建 → 结构化结论 → 继续 / 调整
```

## 三、患者详情 360（11 个子 Tab）

| # | Tab | 核心内容 |
|---|-----|---------|
| 1 | **数字孪生融合分析** | 健康分卡片（0–100 分 + 风险等级 + 推演摘要 + 影响因素标签）；孪生分析依据（3–4 个核心指标卡）；共病风险传导链路（激活 / 待补 / 正常三态，动态叙事文案）；孪生管理建议（复测 / 随访 / 医生建议任务卡）；推荐依据（折叠列表）；后续管理路径（操作入口） |
| 2 | **血糖深度分析** | 数字孪生入口横幅；CGM / 指尖双子 Tab；5 项 CGM 指标卡（TIR / 均值 / CV / TAR / eA1c）；AGP 折线图 + TIR 分布图（7 / 14 / 30 天切换）；指尖 5 项指标卡（空腹 / 餐后 / 达标率 / 波动 / 低糖次数）；AI 分析卡；明细记录分页；症状记录 |
| 3 | **睡眠深度分析** | 数字孪生入口横幅；5 项指标卡（AHI / ODI / 最低血氧 / 体动鼾声 / 报告有效性）；多晚趋势图（指标切换 + 时间窗切换）；当前报告摘要（呼吸事件 + 夜间血氧分段）；历史报告弹层 |
| 4 | **慢阻肺深度分析** | 数字孪生入口横幅；6 项指标卡（均值血氧 / 最低血氧 / ODI / 低氧事件次数 / 低氧时长 / Ts90%）；血氧时序图 + ODI 趋势图（Chart.js）；CAT / mMRC 问卷评分；症状标签；活跃方案治疗摘要；共病重叠综合征卡（OSA+COPD 时展示） |
| 5 | **健康档案** | 5 个信息区：基础信息 / 疾病信息（首诊 + 并发症）/ 用药治疗史 / 生活方式与家族史 / 设备信息 |
| 6 | **健康筛查** | 健康分展示；已确认 / 待确认病种摘要；各病种风险卡（评分 / 等级 / 触发因素）；"确认疾病"操作 |
| 7 | **预警记录** | 该患者预警卡片列表（等级 / 来源 / 状态） |
| 8 | **管理方案** | 该患者方案卡片列表 + 新建方案草稿按钮 |
| 9 | **随访计划** | 该患者随访卡片列表 + 创建随访按钮 |
| 10 | **医生建议** | 已发建议卡片列表（标题 / 来源 / 发送时间 / 状态 / 内容）+ 发送医生建议按钮 |
| 11 | **时间轴** | 全事件时序列表（类型 / 时间 / 操作人） |

## 四、弹层 / 抽屉（已实现 14 个）

| 触发场景 | 类型 | 主要内容 |
|---------|------|---------|
| 处理预警 | Modal | 结论输入、生成随访 / 调整方案选项 |
| 新建管理方案 | Modal | 患者 / 病种 / 周期选择、3 种创建方式（推荐 / 复制 / 空白） |
| 确认下发方案 | Modal | 校验结果（必填模块完成数 + 警告）、冲突处理、患者预览链接 |
| 患者知晓确认 | Modal | 方案标题摘要 + 标记知晓按钮 |
| 驳回推荐方案 | Modal | 驳回原因单选 + 备注 |
| 停用方案 | Modal | 停用原因单选 + 备注 |
| 患者反馈处理 | Modal | 反馈内容 + 医生建议操作 |
| 关闭模块确认 | Modal | 影响范围预警 + 确认按钮 |
| 患者可见内容预览 | Modal | 小程序卡片风格方案内容预览 |
| 历史睡眠报告 | Modal | 报告列表可选行 |
| 创建随访 | Drawer | 基础信息（类型 / 方案 / 原因）+ 时间安排（快捷时间 + 日期时间 / 方式 / 负责人）+ 患者准备（材料清单 + 量表区） |
| 发送医生建议 | Drawer | 来源信息 + 标题 / 正文字段（含字数校验） |
| 创建复测任务 | Drawer | 指标多选（最多 3 项）+ 场景 / 原因 / 频次 / 时长 + 患者指导语 + 起止日期 |
| 编辑方案模块 | Drawer | 按模块类型渲染字段编辑器（目标 / 指标 / 用药 / 症状 / 生活方式 / 随访规则 / 预警规则）+ 恢复推荐值按钮 |
